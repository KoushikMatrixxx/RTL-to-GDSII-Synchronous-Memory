# TCL Scripts

This folder contains TCL scripts used during the RTL-to-GDSII Physical Design flow in Cadence Innovus.

## Scripts

- floorplan.tcl – Floorplan creation and core setup
- power_plan.tcl – Power ring and power stripe generation
- placement.tcl – Standard cell placement and optimization
- cts.tcl – Clock Tree Synthesis (CTS)
- route.tcl – Global and detailed routing

These scripts automate different implementation stages and help achieve a successful physical design flow.
