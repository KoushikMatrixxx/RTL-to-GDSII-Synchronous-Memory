# Reports

This folder contains implementation reports generated during the RTL-to-GDSII Physical Design flow using Cadence Innovus.

These reports summarize utilization, timing, power, routing quality, physical verification, and implementation statistics obtained at different stages of the design flow.

## Available Reports

- Design Utilization Summary
- Area Summary
- Power Summary
- Routing Congestion Report
- Post-CTS Setup Timing
- Post-CTS Hold Timing
- Post-Route Setup Timing
- Post-Route Hold Timing
- Design Rule Violation (DRV) Report
- Detailed Timing Path
- Physical Verification (DRC)
- Connectivity Verification
- Metal Density Check
- Final Innovus Summary

These reports demonstrate the successful implementation and verification of the synchronous memory design.
